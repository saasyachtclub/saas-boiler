## 🎯 **Quick Actions**

### 🚀 **Get Started**
```bash
# Clone the repository
git clone https://github.com/saasyachtclub/saas-boiler.git

# Install dependencies
cd saas-boiler && pnpm install

# Start development server
pnpm dev
```
